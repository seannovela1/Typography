Web Development

History

Web development has been a growing industry since the mid-1990s. The growth of this industry is being driven by businesses wishing to use their website to sell products and services to customers. There are many open-source tools for web development such as BerkeleyDB, GlassFish, LAMP (Linux, Apache, MySQL, PHP) stack and Perl/Plack.

Technologies

Hypertext Markup Language (HTML)

HTML is the language used to create web pages It is a markup language that web browsers use to interpret and compose text, images, and other material into visual or audible web pages. .

Cascading Style Sheets (CSS)

CSS is a style sheet language used for describing the presentation of a document written in a markup language like HTML. CSS is a cornerstone technology of the World Wide Web, alongside HTML and JavaScript

Development Tools

Web developers use a variety of tools depending on what part of the production process they are involved in. These tools are updated over time by newer standards and software but the principles behind them remain the same.

Web developers use both client-side and server-side tools to create web pages. Client-side tools include HTML, CSS, and JavaScript, while server-side tools may include PHP, ASP.NET, and Node.js.
